Exemplo de Atualização de Referências

Como Atualizar os Templates

Este arquivo demonstra como as referências a arquivos estáticos devem ser atualizadas após a reorganização do projeto.

Referências CSS:

<link rel="stylesheet" href="styles.css">


<link rel="stylesheet" href="{{ url_for('static', filename='css/styles.css') }}">
Referências JavaScript:

<script src="script.js"></script>


<script src="{{ url_for('static', filename='js/script.js') }}"></script>
Referências a Imagens:

<img src="imagem.jpg">


<img src="{{ url_for('static', filename='images/imagem.jpg') }}">
Navegação Entre Páginas
Links para outras páginas:

<a href="config.html">Configurações</a>


<a href="{{ url_for('config') }}">Configurações</a>
