# Pasta de Imagens de Exemplo

Adicione suas imagens de visão top-down nesta pasta para treinar o modelo DeepLabCut.

Formatos suportados:
- JPG
- JPEG
- PNG

Recomendações:
1. Use imagens de alta qualidade e boa iluminação
2. Inclua diferentes posições de cotovelo e pulso
3. Adicione pelo menos 5-10 imagens para um bom treinamento inicial

Após adicionar as imagens, execute o script de treinamento:
```
python ../treinamento.py
```
